
## Toy scala

스칼라에 익숙해지기 위한 토이프로젝트.


## 목표

1. [reactive programming](https://github.com/ReactiveX/RxScala)을 경험해보자
2. 간단한 API서버를 만들어보자.
3. 모니터링이 가능한 API서버를 만들어보자.
4. 꾸준히 커밋해보자.

## run

```
sbt assembly
java -jar ./target/scala-2.12/toy-scala-assembly-0.1.jar
```
